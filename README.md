# tarm
a tar extractor witth many useful features, such as auto deleting files after extract

INSTALL GUIDE
(linux only)

run this:
```
sudo rm -rf ~/tarm /usr/local/bin/tarm && git clone https://github.com/spaceguythe1/tarm.git && sudo mv ~/tarm/tarm /usr/local/bin && sudo chmod +x /usr/local/bin/tarm && sudo rm -rf ~/tarm
```
(command can also be used to update)
its installed.

USAGE

tarm (the file you want to extract, MUST END IN .tar.gz or will just delete file)

OR

tarm --version (or -v)
prints version (what else would it do?)

i only have support for .tar.gz rn.
